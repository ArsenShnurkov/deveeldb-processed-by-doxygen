Detailed Description

Represents an event fired at a given modification event (either INSERT, DELETE or UPDATE) at a given time (BEFORE or AFTER).

Event triggers can be of two different main categories:

  • Callback, called for every modification event of the given type on any resource and notified only to the user client
  • Procedure, that executes a procedure (either external, internal or defined as the body of the trigger)
