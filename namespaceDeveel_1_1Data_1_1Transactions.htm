The mode applied to a lock over a resource during a transaction.

Enumerator
None 

No specific locking was applied to the resource

Exclusive 

An exclusive lock was applied to the resource: this can be accessed only by the transaction that owns the lock or by child transactions.

Shared 

A lock that is shared between transactions, allowing read of data on locked objects, but preventing any write.
