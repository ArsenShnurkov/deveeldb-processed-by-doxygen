Detailed Description

Defines the contract for the business managers of database objects of a given type.

Implementations of this interface will allow the system to:

  • Create the required tables into the system at initialization
  • Create database objects given a IObjectInfo specification
  • Alter an existing object
  • Assess the existence of a given object by name
  • Get the instance of an object by its unique name
  • Delete an object from the system
  • Normalize the name of an object
